# 폴리큐브 백엔드 개발자 코딩 테스트

**(주)폴리큐브**에서 시행하는 백엔드 개발자 코딩 테스트용 리포지토리입니다.

## 1. 시작하기

### 1.1. 개발 환경

- OpenJDK 17
- Spring Boot 3.2.1

### 1.2. 라이브러리

- Spring Web
- Lombok
- H2 Database ( ID : pc, PW : 2024 )
- 그 외 필요한 라이브러리는 `build.gradle`에 추가하시면 됩니다.

### 실행 방법

#### 실행
```shell
./gradlew bootRun
```

#### 테스트
```shell
./gradlew test
```
